# Mis Ahorros JP 💰

Aplicación simple y práctica para **registrar ingresos, gastos y ahorros** hecha especialmente para Juan Pablo.

**Versión 1.0** - Primera versión funcional

---

### 🎯 ¿Para qué sirve?

- Registrar cuando te llega un pago (sueldo, etc.)
- Registrar gastos diarios con categorías
- Ver tu **saldo actual** en tiempo real
- Llevar control de categorías como: compras compulsivas, comida chatarra, ahorro estufa a pellet, etc.
- Funciona completamente **offline** en tu celular

---

### 🛠 Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (datos guardados en tu navegador/celular)
